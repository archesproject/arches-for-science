{% trans "New Project Workflow" %}

This workflow creates a new Project resource instance. In Arches for Science (AfS), a Project is a set of activities and tasks revolving around an object(s) (i.e. Physical Thing(s)) to accomplish a specific set of goals over a period of time. AfS projects typically involve data collection and interpretation together with reporting outcomes.

The new Project created by this workflow will be used to link together other resource instances (such as Physical Things, Persons, Instruments, Observations, Sampling Activities, etc.) that relate together through the studies and analyses conducted in the Project.

This workflow will ask you to provide the following information:

Required Information
  • Project name
  • Scale of Project (initiative, project, or research task)
  • Earliest start date
  • Project Team Members (existing or new Person resource instances)
  • Physical Things (studied by the Project)
Optional Information
  • Statement (textual description)
Workflow Outcomes

Completion of this workflow will result in the creation of a new Project resource instance and a new Collection resource instance. The project's collection will be used to organize the Physical Things resource instances involved in the Project.
