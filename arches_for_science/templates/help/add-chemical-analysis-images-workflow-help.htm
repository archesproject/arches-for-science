{% trans "Add Chemical Analysis Images Workflow" %}

This workflow enables you to upload images for use in Image Services to associate with an "Observation" resource instance. Alternatively, you can annotate an existing Image Service to provide information that helps describe an Observation resource instance. In doing so, you can provide explanatory graphical information derived from previously uploaded datafiles.

Required Information
  • Physical Thing (subject of the dataset)
  • Observation (resource instance)
  • Image Service (either already existing or created by uploading an image file)
Workflow Outcomes

This workflow will associate annotations on an Image Service with information about an Observation resource instance.
