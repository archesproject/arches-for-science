{% trans "Sample Taking Workflow" %}

This workflow creates a Sampling Activity resource instance that describes how and where a new sample (Physical Thing resource instance) was obtained from another sampled Physical thing resource instance.

Use image services in the 3rd and 4th step of this workflow to describe the location of where the sample was obtained.

Required Information
  • Project
  • Physical Things (studied by the Project)
Optional Information
  • Samplers (people conducting the sampling)
  • Sampling Date
  • Sampling Activity Type
  • Sampling Motivation (textual description)
  • Sampling Technique (textual description)
  • Image Service and Sampled Region (of sampled item)
Workflow Outcomes

Completion of this workflow will create a new Sampling Activity resource instance and a new sample (Physical Thing) resource instance. It will record the relationships between the sample, Sampling Activity, and the Physical Thing resource instance that was the source of the sample.
