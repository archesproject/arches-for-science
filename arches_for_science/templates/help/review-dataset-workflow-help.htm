{% trans "Review Dataset Workflow" %}

This workflow does not create or update data, rather it provides a convenient way to find and review 2 dimensional (X-Y) datasets associated with a selected Physical Thing resource instance.

Required Information
  • Physical Thing (subject of the dataset)
  • Dataset to select for review
Workflow Outcomes

This workflow provides an opportunity to review a 2 dimensional (X-Y) dataset provided that dataset has an appropriate file render configured and selected.
