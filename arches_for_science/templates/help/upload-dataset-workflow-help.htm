{% trans "Upload XY Data Workflow" %}

This workflow associates 2 dimensional (X-Y) data with sampling activities, physical things, instruments, and other relevant information. Using this workflow, you can select or define renderers (provided the dataset is in a text-based format that can be parsed).

Use image services in the 3rd step of this workflow to describe a location described by the 2D dataset.

Required Information
  • One or more 2 dimensional (X-Y) data files
  • Project
  • Physical Things (subject of the dataset)
Optional Information
  • Image Service (to define a sampled / studied location)
  • Instrument
  • Observation Type
  • Observation Date
  • Procedure Used (a Textual Work resource instance)
  • Instrument Parameters (textual description)
  • Image Service and Sampled Region (of sampled item)
  • File Render (Note: no preview of the dataset will be available unless an appropriate file render is configured and selected)
Workflow Outcomes

Completion of this workflow will create a new Observation resource instance that documents the uploaded 2D data and associates it with a relevant Physical Thing resource instance.
