{% trans "Chemical Analysis Data Upload Workflow" %}

This workflow associates a large data file (including "data cubes") with sampling activities, physical things, instruments, and other relevant information. While the datafile will be too large and may require special software to process and render, association with other Arches for Science managed information will make it easier to find, manage, and understand.

Required Information
  • One or more data files
  • Project
  • Physical Things (subject of the dataset)
Optional Information
  • Instrument
  • Observation Type
  • Observation Date
  • Procedure Used (a Textual Work resource instance)
  • Instrument Parameters (textual description)
Workflow Outcomes

Completion of this workflow will create a new Observation resource instance that documents the uploaded data file and associates it with a relevant Physical Thing resource instance.

Note: the uploaded datafile will typically be stored in a file system (usually cloud storage) external to Arches for Science.
